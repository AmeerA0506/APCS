CW-Binary to Decimal and Vice Versa conversions
2021-09-23

Bin2Dec
set the value of y to 0. for each integer (int) in the input, if the int number to the right is 1, add the value of 2^(int-1) to y. return y after the loop
Dec2Bin
If the inputted value is not 0, set n to 1 and make a loop so that the remainder of x/2 is placed into the nth character of a string. After that, set x to x//2 and continue. When x == 1, make the last digit 1 and stop. If x==0, set the string to 0.
